## 📚 Теоретичні відомості
**PSoC Creator** — це потужне інтегроване середовище розробки (IDE) від компанії Cypress, призначене для проєктування пристроїв на базі мікроконтролерів PSoC. 

**Основні можливості IDE:**
* Зручне створення та керування проєктами.
* Графічне конфігурування апаратних компонентів (схемотехнічний ввід).
* Автоматична генерація API (програмних інтерфейсів) для налаштованих компонентів.
* Компіляція, прошивка та відлагодження коду.
* Швидкий доступ до великої бази готових рішень (Code Examples).

*У цій роботі реалізовано базовий проєкт: керування станом світлодіода за допомогою фізичної кнопки.*

---

## 🛠 Хід виконання роботи

### 1. Запуск середовища та створення проєкту
* Після запуску PSoC Creator відкрито головне вікно програми.
* Створено новий проєкт. У вікні **Device Selector** як цільовий пристрій обрано мікроконтролер **CY8C4245AXI-483**.

### 2. Додавання та налаштування компонентів
На робоче поле схеми (TopDesign) було додано два компоненти з бібліотеки:
1. **Digital Input Pin** (для зчитування стану кнопки).
2. **Digital Output Pin** (для керування світлодіодом).


**Параметри компонентів:**
* **Кнопка (Name: Button):** * Type: `Digital Input`
  * Drive Mode: `Resistive Pull Up` (підтягувальний резистор)
  * Initial Drive State: `High (1)`
* **Світлодіод (Name: LED):** * Type: `Digital Output`
  * Drive Mode: `Strong Drive`
  * Initial Drive State: `Low (0)`

### 3. Призначення пінів (Pin Mapping)
Відповідно до специфікації налагоджувальної плати **CY8CKIT-042 PSoC 4 Pioneer Kit**, порти введення-виведення було налаштовано наступним чином:
* **RGB-світлодіод:** `P1.6` (Червоний), `P0.2` (Зелений), `P0.3` (Синій).
* **Кнопка (USER BUTTON):** `P0.7`.

## 💻 Програмна реалізація
У файлі `main.c` було написано наступний код для керування периферією:

```c
#include "project.h"

int main(void)
{
    CyGlobalIntEnable; /* Увімкнення глобальних переривань */

    for(;;)
    {
        /* Якщо кнопка не натиснута (стан 1 через Pull-Up) */
        if(Button_Read() == 1)
        {
            LED_Write(0); /* Світлодіод увімкнений */
        }
        else
        {
            LED_Write(1); /* Світлодіод вимкнений при натисканні */
        }
    }
}
```
**Логіка роботи:** За замовчуванням світлодіод світиться. При фізичному натисканні на кнопку на платі — світлодіод вимикається. Після відпускання — вмикається знову.

## 🚀 Інструкція із запуску
**Апаратне та програмне забезпечення:**
* IDE PSoC Creator 4.4
* Плата CY8CKIT-042 PSoC 4 Pioneer Kit
* Кабель для підключення по USB

**Кроки для запуску:**
1. Відкрити проєкт у середовищі розробки.
2. Скомпілювати код, натиснувши **Build** у верхньому меню.
3. Підключити плату до ПК.
4. Прошити мікроконтролер, натиснувши кнопку **Program**.
